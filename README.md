## Project
This is part of my FutureGames C++ with Unreal course.

## Game Guide
This game is an attempt to clone Call of duty Zombies. It's a wave based zombie survival game.
Your goal is to survive for the highest amount of rounds possible and set new highscores. You can survive alone or with friends 1-4 player coop.

If you are playing Co-op. The main player uses Keyboard and the rest need controllers xbox/ps4. Ps5 works if you use an emulator.

### Advice
The zombie hordes eventually scale in size and stats. So make sure to keep upgrading your weapons with the pack-n-punch machine!
Playing with friends introduced the revive mechanic. So if you are 2+ players you can revive another by holding the Interact button down while aiming at them!
